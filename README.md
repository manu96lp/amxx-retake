# [AMXX] Retakes

Retakes is a clone of the popular CS:GO retakes mod made for CS 1.6.

## Requirements

```
- AMX MOD X 1.8.3+
- ReHLDS
- ReGameDLL
- ReAPI
- SQLite
```
